# 🌡️ Air Temperature Time Series Analysis

## 📌 Project Overview
This project focuses on analyzing and forecasting air temperature patterns using **Time Series Analysis** and **Machine Learning** techniques. Historical temperature data is processed to identify trends, seasonality, and temporal patterns, enabling accurate future temperature predictions.

## 🎯 Objective
To analyze historical air temperature data and build a forecasting model that predicts future temperature values using time series techniques.

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Statsmodels

## 📊 Workflow
1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Trend Analysis
5. Seasonality Analysis
6. Time Series Modeling
7. Model Evaluation
8. Temperature Forecasting

## 🔍 Key Analysis Performed
- Temperature trend analysis
- Seasonal pattern detection
- Time series decomposition
- Forecasting future temperature values
- Visualization of historical and predicted temperatures

## 📈 Results
- Identified long-term temperature trends
- Detected seasonal variations in temperature data
- Generated accurate temperature forecasts
- Created visualizations to support data-driven insights

## 🚀 How to Run

### Clone the Repository

```bash
git clone https://github.com/Hruthi2910/Air-temperature-time-series.git
```

### Navigate to the Project Folder

```bash
cd Air-temperature-time-series
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```

### Run the Notebook
Open the notebook file and execute all cells sequentially.
